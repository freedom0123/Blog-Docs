## 一、简介

1.  以 JavaScript 为基础构建的语言
2. 可以在任何支持JavaScript 的平台之中执行
3. 一个JavaScript的超集，扩展类JavaScript，并添加了类型
4. TS 不能被 JS 编译器直接执行，最后需要进行编译成 JS 运行

TypeScript 增加了什么？

- 添加了类型
- 支持 ES 的新特性
- 丰富的配置选项
- 强大的开发工具

## 二、环境搭建

1. 安装Node js

```markdown
npm i -g typescript
```

创建一个ts文件

编写完成之后，通过
